## 使用方法
### Gradle
```groovy
compile 'com.lihaodong.toast:toastlibrary:1.0'
```
### Maven
```groovy
<dependency>
  <groupId>com.lihaodong.toast</groupId>
  <artifactId>toastlibrary</artifactId>
  <version>1.0</version>
  <type>pom</type>
</dependency>
```
